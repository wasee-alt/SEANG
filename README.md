# S.E.A.N.G • ProPlugin (Streamlit Cloud ready)

**วิธีใช้งานบน Streamlit Cloud**  
1) อัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้ขึ้น GitHub (public repo)  
2) ไปที่ https://share.streamlit.io → New app  
3) เลือก repo / branch และตั้ง App file = `app.py` → Deploy  

**ทดสอบในเครื่อง**  
```bash
pip install -r requirements.txt
streamlit run app.py
```
